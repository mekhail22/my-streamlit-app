# Christian Ecumenical Councils Guide

An interactive web application built with **Streamlit** that provides a comprehensive guide to the Christian Ecumenical Councils.

## 🌐 Live Demo
[View the App](https://christian-history-councils.streamlit.app/)

## 📋 Features
- Detailed information about each Ecumenical Council
- Interactive navigation between councils
- Historical context and key decisions
- Clean and responsive user interface

## 🛠️ Tech Stack
- **Python**
- **Streamlit**
- **Pandas** (for data handling)

## 🚀 Run Locally
```bash
# Clone the repository
git clone https://github.com/mekhail22/my-streamlit-app.git

# Navigate to the project
cd my-streamlit-app

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
